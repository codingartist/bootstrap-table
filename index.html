Base HTML

Table options

Tag Description
<table> Wrapping element for displaying data in a tabular format
<thead> Container element for table header rows (<tr>) to label table columns
<tbody> Container element for table rows (<tr>) in the body of the table
<th> Special table cell for column (or row, depending on scope and placement) labels
Must be used within a <thead>
<caption> Description or summary of what the table holds, especially useful for screen readers 
	<table data-type="table" url="/table/load/">
	  <thead>
	    <tr>
	      <th>…</th>
	      <th>…</th>
	    </tr>
	  </thead>
	  <tbody>
	  </tbody>
	</table>

<table> Attributes

Attribute Required Default Description
data-role="" true null Must be for the handles by the table script. Only tables with the data-role="table" attribute will be processed by the table JavaScript, all other tables will be ignored.
url="" true null The url used to load the table, must be set or table will load with error status
size='' false 10

The number of rows that will be displayed on a page. This value is passed top the load script via the size parameters.

Heads up! If additional records are returned from the load script the table will still display them however the paging widget will not take them into account.
parameters='' false '' A JSON formatted string of parameters that will be passed to the load script, Example: parameters='"parameters1":"value1","parameters2":"value2"'

<th> Attributes

Attribute Required Default Description
data-id="" false column number The data id used to map the JSON response to the appropriate column, id null the column number will be used starting with 0. {@see Example}
data-derection="" false ASC The url used to load the table, must be set or table will load with error status
